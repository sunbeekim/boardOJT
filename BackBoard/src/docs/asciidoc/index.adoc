== API 명세

=== 요청 정보
[source,http]
----
POST /api/user/signup
Content-Type: application/json
----

=== 요청 필드
|===
|필드명|타입|필수|설명|제약조건
|email|string|Y|이메일|이메일 형식
|password|string|Y|비밀번호|8~16자, 영문/숫자/특수문자
|nickname|string|Y|닉네임|2~10자, 한글/영문/숫자
|===

=== 응답 정보
[source,json]
----
{
  "success": true,
  "message": "회원가입이 완료되었습니다.",
  "data": 200,
  "error": null,
  "errorPath": null
}
----
